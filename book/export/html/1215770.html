148. What is it?

Love:
What is it
If not the royal road
To Realisation?

Devotion:
What is it
If not the shortcut
To Realisation?

Surrender:
What is it
If not a bud
Of the Realisation-Tree?

From: Sri Chinmoy, The Wings Of Light, Part 3, Agni Press, 1974
