224. Only remedy

        Only remedy
For the delinquent world: austerity.
        Only remedy
For the fallen world: concern.
        Only remedy
For the helpless world: Love.
        Only remedy
For the dying world: Oneness.

From: Sri Chinmoy, The Dance of Life, part 5, Agni Press, 1973
