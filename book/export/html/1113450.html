51

Some are of the opinion
That God-manifestation
  Is an impossibility,
While others think
That God-manifestation
  Is a remote possibility.
Still others think that God-manifestation
  Is a living reality.
I proudly and happily
Belong to the last category.

From: Sri Chinmoy, Seventy-Seven Thousand Service-Trees, Part 1, Agni Press, 1998
