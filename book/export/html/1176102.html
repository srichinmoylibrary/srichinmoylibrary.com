3148 If You Want God

If you want inspiration,
Then choose beauty.

If you want aspiration,
Then choose truth.

If you want realisation,
Then choose devotion.

If you want God,
Then choose surrender.

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 32, Agni Press, 1982
