28. Ever the same again

Ever the same again,
My lost Truth rediscovered.
Ever the same again.

Ever the same again,
My forgotten Self remembered.
Ever the same again.

Ever the same again,
My lost Goal regained.
Ever the same again.

From: Sri Chinmoy, My Flute, Agni Press, 1972
