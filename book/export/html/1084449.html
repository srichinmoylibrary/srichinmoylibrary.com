15. If you really want to see

Don't separate,
Don't divide;
Do unite,
Do reconcile
If you really want to see
God in man.
Don't hate,
Don't destroy;
Do love,
Do build
If you really want to see
Man in God.

From: Sri Chinmoy, The Golden Boat, part 10, Agni Press, 1974
