3734 My Mind's Sincerity-River

My mind's sincerity-river
Shall never run dry.
My heart's purity-river
Shall never run dry.
My life's humility-river
Shall never run dry.
My soul's illumination-river
Shall never run dry.

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 38, Agni Press, 1982
