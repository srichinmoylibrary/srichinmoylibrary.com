51. Duty before pleasure

Duty before pleasure:
If we do that
We can achieve and
Treasure God-Light.
Pleasure before duty:
If we do that
We shall dance death-dance
With death
Before
We sing life-song
With life.

From: Sri Chinmoy, Sound becomes, silence is, Agni Press, 1975
