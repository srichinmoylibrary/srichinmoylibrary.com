32. No time to worry

No time to worry
If you want to serve God.

No time to worry
If you want to love God.

No time to worry
If you need God.

No time to worry
If you really want to be the God of tomorrow.

From: Sri Chinmoy, When God-Love descends, Agni Press, 1975
