172. If I have a crown

If I have a life,
Then it is for the dying earth.

If I have an iota of Bliss,
Then it is for the suffering earth.

If I have a crown,
Then it is for the striving earth.

From: Sri Chinmoy, Europe-Blossoms, Agni Press, 1974
