2041 The Code Of Your Life

If you are a true disciple,
Then the code of your life
Should be
"Renounce."

If you are a true Master,
Then the code of your life
Should be
"Accept."

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 21, Agni Press, 1981
