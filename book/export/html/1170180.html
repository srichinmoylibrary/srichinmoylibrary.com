84. Try to satisfy

Try to satisfy humanity.
You certainly can.
Try to satisfy yourself.
Indeed, this is an impossible task.
Try to satisfy God.
Lo, you have already done it
And you have done it quite well.

From: Sri Chinmoy, Sound becomes, silence is, Agni Press, 1975
