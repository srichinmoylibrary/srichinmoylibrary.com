307 A Marathon Run

A marathon run
Is not fun.
It is an unspeakably
Serious matter.

A marathon run
Turns me into an orphan-life:
No father, no mother,
Helpless and hopeless.

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 4, Agni Press, 1979
