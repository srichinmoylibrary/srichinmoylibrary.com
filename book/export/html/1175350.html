2403 Each New Start

Each serious problem
Intensifies my God-loving heart,
Believe it or not.

Each sad failure
Purifies my God-serving life,
Believe it or not.

Each new start
Strengthens my determination
And
Lengthens my aspiration,
Believe it or not.

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 25, Agni Press, 1982
