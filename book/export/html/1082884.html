Paths to God

What is Life divine
If not a straight path to God?

What is Light divine
If not a sunlit path to God?

What is love divine
If not a straight, sunlit shortcut,
A supreme shortcut
To God?

From: Sri Chinmoy, God-Life: is it a far cry?, Agni Press, 1974
