2672 God's Dinner

Realisation - what is it,
If not the soul's breakfast?

Manifestation - what is it,
If not Heaven's luncheon?

Satisfaction - what is it,
If not God's dinner?

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 27, Agni Press, 1982
