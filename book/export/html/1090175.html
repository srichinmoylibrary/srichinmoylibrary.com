41. If you have to fly

If you have to fly,
Then fly for the highest truth.

If you have to run,
Then run for the brightest light.

If you have to dive,
Then dive for the deepest delight.

From: Sri Chinmoy, Lord, receive this little undying cry, Agni Press, 1975
