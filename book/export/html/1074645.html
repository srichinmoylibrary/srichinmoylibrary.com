82. You must teach

Only one body,
You must teach it.
Only one vital,
You must guide it.
Only one mind,
You must perfect it.
Only one heart,
You must use it.
Only one soul,
You must treasure it.
Only one Goal,
You must become it.

From: Sri Chinmoy, The Dance of Life, part 2, Agni Press, 1973
