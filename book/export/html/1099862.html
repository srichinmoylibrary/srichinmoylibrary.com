27. Never the same again

Never the same again,
Lost peace restored.
Never the same again.

Never the same again,
Lost joy regained.
Never the same again.

Never the same again,
Lost power reborn.
Never the same again.

From: Sri Chinmoy, My Flute, Agni Press, 1972
