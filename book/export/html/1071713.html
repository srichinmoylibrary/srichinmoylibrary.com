January 23

I cannot create Truth. But I can discover Truth.
I cannot create Love. But I can discover Love.
I cannot discover Aspiration. But I can create Aspiration.
I cannot discover Perfection. But I can create Perfection.

From: Sri Chinmoy, Arise! Awake! Thoughts of a yogi, Frederick Fell, Inc., 1972
