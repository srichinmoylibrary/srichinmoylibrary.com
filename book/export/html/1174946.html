2003 What Is An Accident?

What is an accident
If not an incident?

What is an incident
If not an experience?

What is an experience
If not the beginning
Of my realisation?

From: Sri Chinmoy, Ten Thousand Flower-Flames, Part 21, Agni Press, 1981
