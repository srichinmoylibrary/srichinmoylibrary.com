752 New Creation, New Creator...

NEW CREATION, NEW CREATOR

I don't want beauty,
I want power.

I don't want power,
I want love.

I don't want love,
I want light.

I don't want light,
I want God.

I don't want God,
I want a new creation
With the new creator: C.K.G.

From: Sri Chinmoy, Europe-Blossoms, Agni Press, 1974
