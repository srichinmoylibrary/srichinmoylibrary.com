38. Only that

Only what is permanent
   Is free.

Only what is free
   Is divine.

Only what is divine
   Is fulfilling.

Only what is fulfilling
   Is life-giving.

Only what is life-giving
   Is God-becoming.

From: Sri Chinmoy, The Golden Boat, part 2, Agni Press, 1974
